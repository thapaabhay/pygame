# Pygame Demo

## Description
A simple Python program built with Pygame.

## Requirements
- Python 3.11 or higher
- Pygame library

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/thapaabhay/pygame.git
   cd pygame
   ```
2. Install the dependencies:
   ```bash
   pip install pygame
   ```

## Usage
Run the main script:
```bash
python main.py
